**Predict CSAT Score (1-5) in Real-Time using Text Remarks + Metadata**

---

### 📊 Project Highlights

- **Accuracy**: 91.2%  
- **Macro F1 Score**: 0.887  
- **1-Star Recall**: 0.94 (Critical for business)  
- **Interactive Web App** built with Streamlit  
- **12 Business Visualizations** for insights  
- **Real-time Prediction** for support tickets

---

### ✨ Features

- 📝 Advanced Text Preprocessing (TF-IDF + SVD)
- 🧠 XGBoost Multi-class Classification
- 📈 Real-time CSAT Prediction
- 📊 12 Insightful Visualizations
- ⚠️ Risk Alert System (Low CSAT Prediction)
- 🔄 Auto-escalation recommendations
